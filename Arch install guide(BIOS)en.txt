INSTALLING ARCH LINUX

<We make a bootable USB flash drive using dd in Linux>

$ dd if=/путь/archlinux.iso of=/dev/диск status=progress


<Or using Rufus, Unetbootin, or others on Windows>

We boot from the flash drive and begin the installation.


<Checking network connectivity>

$ ping archlinux.org

for wireless connection use the wifi-menu program

$ wifi-menu

DISK MARKING

/dev/sda1 - boot - ext2 - 500M
/dev/sda2 - root - ext4 - 30G+
/dev/sda3 - swap - linuxswap - RAM amount
/dev/sda4 - home - ext4 - remainder

<Create partitions using cfdisk>

<Format sections>

$ mkfs.ext2 /dev/sda1 -L boot
$ mkfs.ext4 /dev/sda2 -L root
$ mkswap /dev/sda3 -L swap
$ mkfs.ext4 /dev/sda4 -L home

<Mount sections>

$ mount /dev/sda2 /mnt
$ mkdir /mnt/{boot,home}
$ mount /dev/sda1 /mnt/boot
$ mount /dev/sda4 /mnt/home
$ swapon /dev/sda3

<Package Installation>

$ pacstrap -i /mnt base base-devel linux 
linux-firmware dhcpcd nano intel(or amd)-ucode reflector grub


PRIMARY SYSTEM SETUP

<Fstab generation>

$ genfstab -p /mnt >> /mnt/etc/fstab

<Login to the installed system>

$ arch-chroot /mnt

<Add locale to the system>

$ nano /etc/locale.gen

Uncomment (remove # first) the line # en_US.UTF-8 UTF-8

$ locale-gen

<Bootloader installation>

$ grub-install /dev/sda

$ grub-mkconfig -o /boot/grub/grub.cfg

<Set a password for root>
$ passwd

exit chroot
$ exit

<Unmount the partitions>

$ umount -R /mnt

and reboot the PC 

$ reboot

AFTER RESET

<Set hostname>

$ echo name >> /etc/hostname

<Time zone set>

$ timedatectl set-timezone Europe/Kiev


<Bootable RAM disk update>

$ mkinitcpio -p linux


<Pacman setup>

$ nano /etc/pacman.conf

Uncomment (remove #) the lines:

#[multilib] 
#Include = /etc/pacman.d/mirrorlist

<Add user>

$ useradd -m -g users -G wheel,audio -s /bin/bash name

and set a password

$ passwd name

<Network configuration>

$ systemctl enable dhcpcd
$ systemctl start dhcpcd

for wireless network use wifi-menu

$ wifi-menu


<Package Updates>

$ pacman -Syy
$ pacman -Su

<Install and configure sudo>

$ pacman -S sudo
$ nano /etc/sudoers

uncomment the line #% wheel ALL = (ALL) ALL

<Setting yay to use AUR>

$ sudo pacman -S git
$ git clone https://aur.archlinux.org/yay.git
$ cd yay 
$ makepkg -si

Done!
